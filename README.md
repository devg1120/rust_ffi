-------------------------------------------------------------  ./test02_shared/a.c

```c
#include <stdio.h>

void func_a() {
  printf("This is func_a!\n");
}
```

--------------------------------------------------------  ./test02_shared/build.sh

```sh
 gcc -shared a.c -o liba.so -fPIC
 nm liba.so
 rustc -o test_rs.exe main.rs -L. -la
 export LD_LIBRARY_PATH=.:$LD_LIBRARY_PATH
 test_rs.exe
```

---------------------------------------------------------  ./test02_shared/main.rs

```rust
extern "C" {
    fn func_a();
}

fn main() {
    unsafe {
        func_a();
    }
}
```
